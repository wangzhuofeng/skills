# Person Note Linker

## 目标

- 在原始想法笔记中把人名改为 Obsidian 链接 `[[人名]]`
- 若人物首次出现，创建 `人物/<人名>.md`

## 处理步骤

1. 识别明确人名（排除群体称呼，如“大家/我们/老师们”）。
2. 在 `人物/` 目录中查找同名笔记：
   - 存在则只在原文替换为 `[[人名]]`。
   - 不存在则创建 `人物/<人名>.md`。
3. 新建人物笔记 frontmatter 参考现有格式：
   - `tags:` 为空
   - `up:` 为空
   - `Time Created` 与 `Time Modified` 设为当前时间
   - `Full Name: <人名>`
   - `categories` 列表包含 `[[人物]]`
4. 新建人物正文追加一行来源记录：
   - 例如：`YYYY/MM/DD: 出现在 [[来源笔记]]。`

## 链接规则

- 原文保持不改写，仅将人名替换为 `[[人名]]`。
- 不添加 `#人物` 标记，避免格式冲突。
